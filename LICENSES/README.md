# Bundled third-party license texts

This directory holds the exact license text of any third-party code
redistributed inside RepoWrangler.

It is currently empty (besides this note) because no upstream source code has
been copied — both upstream projects are conceptual inspiration only. If a
future change copies code, add the upstream license text here (for example
`LICENSES/MIT-git-pull-request-dashboard.txt`) in the same change, and update
`credits.yaml` and `THIRD_PARTY_NOTICES.md`.

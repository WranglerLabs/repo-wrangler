# Support

RepoWrangler is an open-source project maintained by one person with assistance
from Codex. Support is community-style and best effort.

## What to expect

- The latest tagged release is the only supported version.
- Bugs and documentation problems may be filed as GitHub issues with the
  version, deployment method, reproduction steps, expected behavior, actual
  behavior, and sanitized logs.
- There is no paid support, on-call coverage, guaranteed response time, uptime
  commitment, or service-level agreement.
- Operators remain responsible for their cloud accounts, provider credentials,
  backups, availability, monitoring, costs, and disaster recovery.
- Provider and hosting-platform outages or account problems must be handled
  through that provider's support channel.

Security vulnerabilities must use GitHub private vulnerability reporting as
described in `SECURITY.md`, not a public issue.

Clear fixes and documentation improvements are welcome through pull requests.

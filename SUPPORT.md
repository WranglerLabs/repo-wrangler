# Support

RepoWrangler is an open-source project maintained on a best-effort basis.

- **Bugs and feature requests:** open a GitHub Issue with reproduction steps
  or a clear use case.
- **Questions and ideas:** GitHub Discussions (once enabled) or an issue.
- **Security problems:** see `SECURITY.md` — never a public issue.

There is no commercial support or SLA. Operators deploy their own instance on
their own Cloudflare account and own their own GitHub App credentials.

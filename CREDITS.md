# Credits

RepoWrangler exists because other people solved adjacent problems first and
shared their work.

## Inspiration

- **[GitactionBoard](https://github.com/otto-de/gitactionboard)** (Apache-2.0,
  OTTO and contributors) — the idea that a wall of workflow health across many
  repositories is a product, not a script; workflow reliability metrics and
  build-monitor filtering concepts.
- **[Git Pull Request Dashboard](https://github.com/AKharytonchyk/git-pull-request-dashboard)**
  (MIT, Artsiom Kharytonchyk) — multi-organization PR aggregation and
  normalization concepts.

Neither project's source code has been copied. See `THIRD_PARTY_NOTICES.md`
and `credits.yaml` for exact provenance, and the in-product **About & Credits**
page for the same data at runtime.

## Major dependencies

Hono, React, React Router, TanStack Query, Zod, Vite, Vitest, Wrangler, and
the Cloudflare Workers platform.
